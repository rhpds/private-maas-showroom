= Game Time

Now that your AI assistant is wired up, it's time to get your hands dirty - by building a simple game using Continue's help. In this module, you're not solving a business-critical problem… yet! You're exploring, experimenting, and building confidence with AI-assisted coding through something fun: a simple game.

== Choose a Game to Build

Pick **one** of the following lightweight game projects, available in both Python and Bash:

. **Rock Paper Scissors**: Play rock, paper, scissors against the computer!
. **Word Scramble**: The program scrambles a word and the user has to guess what it is.
. **Simple Quiz Game**: The program asks a few trivia questions and then gives a final score.

You only need to pick one - but feel free to try more if you have time.

== Open Your Starter File

From the file explorer, navigate to the `game_starters` directory:

image::game-starters.png[width="50%"]

This folder contains a folder for each game described above. Within each game folder are two files: one for Python code, one for Bash. They contain the same instructions, so just pick the game and language you prefer.

=== Understanding the Starter File Structure

Each game starter file is organized into **three sections** (all as comments):

1. **🎯 Initial Prompt Section** - A ready-to-use prompt you can copy and paste directly into the AI chat interface to generate your first working version of the game

2. **🛠️ AI Assistance Techniques** - Instructions on how to use AI assistance to refine, debug, and improve your code

3. **💡 Enhancement Ideas Section** - Suggestions for features you can ask the AI to add once your basic game is working

=== How to Use the Starter Files

Follow this workflow:

**Step 1: Read the entire file**

Open your chosen game file and read through all the commented instructions to understand what you'll be building.

**Step 2: Generate your first version**

Copy the prompt from the first section (the section that starts with "Create a..." and includes a prompt in quotes) and paste it into the Continue chat interface in the left sidebar. The Nemotron model will generate working code for you.

IMPORTANT: Remember, the Continue chat needs to be set to `Chat` mode and not Agent or Plan modes.

image::add-code-to-chat.png[]

**Step 3: Add the code to your file**

Copy the AI-generated code from the Continue chat extension and paste it into your game file, **below all the commented instructions**. Keep the instructions as comments - they'll help you remember what to do next!

You may also use the `Insert Code` button. Ensure your cursor is at the right point in the file for insertion.

image::add-code-to-file.png[]

IMPORTANT: Keep all instructional comments in the file. Only add actual game code below them. This way you can always refer back to the suggestions.

**Example of what your file should look like:**

[source,python]
----
# === 🎯 SIMPLE QUIZ GAME - PYTHON VERSION ===
#
# Start by asking Continue's chat:
# "Create a Python quiz game that..."
# (all the instructional comments stay here)

# Your actual working code starts here:
score = 0
print("Welcome to the Quiz Game!")
# ... rest of your generated code ...
----

**Step 4: Test your game**

Now let's run your game and make sure it works!

Save your file (kbd:[Cmd+S] on Mac or kbd:[Ctrl+S] on Linux).

Open a terminal in VS Code by navigating to `Terminal` → `New Terminal` from the hamburger menu, or press kbd:[Ctrl+`].

image::new-terminal.png[]

Navigate to your game's folder in the terminal. For example (substitute the correct path to your game file):

[source,console]
----
cd ./coding-exercises/game_starters/rock_paper_scissors/
----

Now run your game based on the language you chose:

NOTE: Substitute the placeholder file names for your actual file.

**For Python (.py files):**

[source,console]
----
python your_filename.py
----

**For Bash (.sh files):**

First, make the file executable (you only need to do this once):

[source,console]
----
chmod +x your_filename.sh
----

Then run it:

[source,console]
----
./your_filename.sh
----

Play your game! If it works, congratulations - you just built something with AI assistance!

**If you see errors:**

* **"command not found" or "permission denied"**: Double-check the filename is spelled correctly, ensure you're in the correct directory (use `ls` to list files), or for .sh files try `bash yourfile.sh`
* **Other errors**: Copy the error message and paste it into the Continue chat - ask "I got this error, can you help me fix it?" The AI can help you debug!

**Step 5 (Optional): Enhance your game**

Once your basic game is working, you can make it better! This step is optional but fun.

Look at the enhancement ideas in the comment sections of your starter file. Use the AI assistance techniques below to add features, improve code quality, or experiment with new ideas.

== Using AI Assistance for Enhancement

If you want to enhance your game (Step 5), here's how to use your AI assistant effectively to iterate and improve your code.

Click the `Continue` extension tab on the left-hand side of the VS Code window to access the chat interface. You can toggle between the file `Explorer` and `Continue` tabs as needed.

image::toggle-sidebar.png[]

=== Two Ways to Get AI Help

Your AI assistant offers two ways to help with your code:

**1. Chat Interface** - Type questions directly into the AI chat sidebar:

* "How can I add more questions to my quiz?"
* "Explain what this code does"
* "How do I make this more efficient?"

**2. Interactive Code Assistance** - Highlight specific code and get targeted help:

To use this approach, highlight the code you want to work with, then right-click and select `Continue` from the pop-up menu (or use kbd:[Cmd+Shift+P] / kbd:[Ctrl+Shift+P] and search for Continue):

image::continue-popup.png[width="50%"]

**AI assistance options:**

* **Add to Chat** - Shares the highlighted code with the AI so you can ask questions about it (e.g., "How does this section work?" or "Can you suggest improvements?"). We used this function already to copy the prompt text from the file into the chat.

* **Add to Edit** - Opens an Edit-mode where you describe changes you want the AI to make to the highlighted code (e.g., "add error handling" or "make this function accept any number of questions")

TIP: A typical AI-assisted workflow: use chat to generate initial code, then use "Add to Edit" on specific sections to refine them. Use "Add to Chat" when you want to understand code or get suggestions before making changes.
