= Troubleshooting

This page provides solutions to issues you may encounter during the workshop.

[#module-03-issues]
== Module 03: Agentic AI Issues

=== MCP Server Authorization Failures

* Authorization is session-based. If you refresh the playground, you must re-select the Rocket.Chat MCP server
* Click the MCP server dropdown, select Rocket.Chat, and click "Save" to reauthorize

=== Message Posting Failures

* Verify you're using the exact channel name: `#workshop`
* Check that the Rocket.Chat MCP Server is still authorized
* If errors persist, try a simpler command first: "List all Rocket.Chat channels" to verify connectivity

=== Playground Issues

* If you receive a timeout error, resend the message
* If authorization or connectivity issues persist, refresh the playground page and re-authorize the Rocket.Chat MCP server

== Getting Additional Help

If you encounter an issue not covered here:

1. **Review error messages carefully** - They often contain specific hints about what's wrong
2. **Ask workshop facilitators** - They can help diagnose cluster-specific issues
3. **Refresh and retry** - Many issues resolve by refreshing the page and re-authorizing MCP servers
